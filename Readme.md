# Roman Number Game

Простая игра на Python, в которой нужно переводить числа между римской и обычной системой счисления.

## 🎮 Как играть

После запуска программы выбери один из режимов (введи 1, 2 или 3):

1. Перевод из римской системы в число
2. Перевод числа в римские цифры
3. Выход

После этого тебе будет предложено случайное число.
Твоя задача — ввести правильный ответ (перевести число в нужный формат).


## ⚙️ Установка и запуск

### Запуск через Python

1. Установи зависимости:

```
pip install -r requirements.txt
```

2. Запусти программу:

```
python roman_game.py
```

### Запуск через .exe

Скачай файл `roman_game.exe` из раздела Releases и просто запусти его.
